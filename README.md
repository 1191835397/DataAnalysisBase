# DataAnalysisBase

本地 **A 股全市场智能监管与分析平台**（设计 v0.2.0）

- 全 A 股 ~5000+ 定时快照与自动化监管
- 行业分类、热力图、异动告警
- 重点股更高频跟踪、多源融合、AI 研报
- 本地 Web 仪表盘（React + FastAPI + DuckDB）

## 当前状态

**设计阶段已完成，代码尚未开始。** 详见 [docs/](docs/) 目录。
