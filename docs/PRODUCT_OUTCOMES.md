# 最终实现效果说明

> 版本 v0.2.0 | 关联：[ARCHITECTURE.md](./ARCHITECTURE.md) · [UI_DESIGN.md](./UI_DESIGN.md) · [ROADMAP.md](./ROADMAP.md)

## 1. 产品定位（v0.2）

> **打开本地 Web 仪表盘 → 看见全 A 股市场与行业变化 → 自动监管告警推送 → 重点股深度跟踪与多源融合 AI 分析。**

最终形态是 **本地 A 股全市场智能监管与分析平台**，不是「预测涨停的炒股神器」。

### 双层体验

| 层级 | 你看什么 | 更新频率 |
|------|----------|----------|
| **全市场** | 5000+ 股票、行业热力、涨跌停、异动告警 | 30 分钟快照 |
| **重点股** | K 线、对账、财务、AI 研报 | 5 分钟 + 按需 |

## 2. 本地 Web 仪表盘（核心交付）

启动后访问 `http://localhost:5173`，详见 [UI_DESIGN.md](./UI_DESIGN.md)。

### 2.1 市场总览页

```text
┌─ A 股市场总览 ──────────────── 更新于 10:30 ─┐
│ 上证 +0.8%  深证 +1.2%  创业板 +1.5%            │
│ 上涨 3120 | 下跌 1890 | 涨停 42 | 跌停 5       │
│ 成交额 8920 亿                                 │
├─ 今日告警摘要 ─────────────────────────────────┤
│ 🔴 涨停 42 只  🟡 放量异动 18 只  🔵 行业异动 3 个 │
└────────────────────────────────────────────────┘
```

### 2.2 行业板块页

- 行业涨跌幅热力图（Treemap）
- 行业排行表：白酒 +2.1%、电池 -0.8% …
- 点击行业 → 成分股列表

### 2.3 全市场股票页

- **~5000 只股票**表格：代码、名称、行业、现价、涨跌幅、成交额、量比
- 搜索、排序、筛选（涨幅榜 / 涨停池 / 放量）
- 虚拟滚动，流畅浏览

### 2.4 监管告警页

```text
10:30  🔴 600519.SH 贵州茅台 涨停
10:30  🟡 300750.SZ 宁德时代 放量异动 量比 2.3
10:00  🔵 行业-电池 整体上涨 2.4%
```

- WebSocket 实时追加新告警
- 按级别 / 行业 / 类型筛选
- 点击跳转个股详情

### 2.5 重点股专区

```text
┌─ 自选 ─────────────────────────────────────┐
│ 600519 茅台  -0.8%  ✓对账  5min前更新      │
│ 300750 宁德  +2.1%  ⚠PE差1.7%  3min前更新   │
└────────────────────────────────────────────┘
[生成研报] [查看对账]
```

### 2.6 个股详情页

**所有股票**：K 线、当日快照曲线、行业、告警历史

**重点股额外**：估值对账、财务趋势、新闻、AI 研报 Tab

## 3. 日常使用方式（v0.2）

```text
盘中（自动）
  → 每 30min 全市场快照 + 监管规则求值
  → 每 5min 重点股深度同步
  → 新告警 WebSocket 推送到仪表盘

用户操作
  → 浏览器打开仪表盘浏览全市场 / 行业
  → 告警页发现异动 → 点击进入个股
  → 重点股专区 → 生成 AI 研报 / 查看对账

收盘后（自动）
  → 日 K 增量、行业分类刷新
```

## 6. 分阶段可见结果

| 阶段 | 你能看到什么 |
|------|--------------|
| **设计阶段（当前）** | 完整文档，无运行界面 |
| **Phase A** | 浏览器：市场总览 + 行业 + 5000 股列表 |
| **Phase B** | + 告警流、行业热力图、WebSocket |
| **Phase C** | + 重点股专区、个股 K 线详情 |
| **Phase D** | + 对账面板、AI 研报 |
| **Phase E** | + 公告 RAG、推送、港股 |

| 里程碑 | 说明 |
|--------|------|
| **第一个能用的界面** | Phase A：打开浏览器即见全市场 |
| **监管平台成型** | Phase B：自动监听变化 + 告警 |
| **完整形态** | Phase D：监管 + 重点股深度 + AI |

## 7. 会有 / 不会有

| 不会有 | 会有 |
|--------|------|
| 秒级全市场 tick | 30min 全市场快照 + 5min 重点股 |
| 5000 只全自动财报 | 重点股财报 + 按需加载 |
| AI 预测涨停 | 规则化异动检测 + AI 解读 |
| 自动下单 | 本地 Web 看板 + 研究辅助 |
| 依赖云端 SaaS | 本地 DuckDB + 本地前端 |

## 8. 与 v0.1 文档关系

v0.1 以 CLI + Markdown 自选研究为主；v0.2 在此基础上增加 **全市场层 + Web 仪表盘 + 监管引擎**。融合、对账、AI 原则不变，**适用范围收窄到重点股**。
